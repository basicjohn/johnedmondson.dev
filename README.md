## Installation

Install my-project with npm

```bash
  npm install my-project
  cd my-project
```

## Run Locally

Clone the project

```bash
  git clone https://github.com/basicjohn/johnedmondson.dev.git
```

Go to the project directory

```bash
  cd johnedmondson.dev
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
